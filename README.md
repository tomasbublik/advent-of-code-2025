# Advent of Code 2025

Moje řešení [Advent of Code 2025](https://adventofcode.com/2025) v Kotlinu.

## Struktura projektu

- `src/` - zdrojové kódy řešení
- `src/DayXX.kt` - řešení pro daný den
- `src/DayXX.txt` - vstupní data
- `src/DayXX_test.txt` - testovací data
- `src/Utils.kt` - pomocné funkce

## Spuštění

Projekt používá Gradle. Pro spuštění řešení konkrétního dne:

```bash
./gradlew run
```

Nebo pomocí přiloženého skriptu:

```bash
./run_day01.sh
```
